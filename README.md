# 🛠️ My offline tools

A collection of fast, clean, and useful developer tools all running directly in your browser.

- ❌ No API calls
- ❌ No tracking
- ❌ No data collection
- ✅ Everything happens locally

---

## ✨ Features

- ⚡ Fast and lightweight
- 🧩 Simple, focused tools
- 🎯 Clean and functional UI
- 🔒 Runs entirely client-side (no data is sent anywhere)

---

## 🧰 Tools

- 🔍 Text Compare (Diff)
- 🔐 JWT Decoder
- 📝 Private Notebook (Synced to local storage)

---

## 🚀 Tech Stack

- Svelte + Vite
- Tailwind CSS
- GitHub Pages

---

## 💻 Development

```bash
npm install
npm run dev
```
